# IoT WiFi & Bluetooth Controlled Car

An IoT project implementing a remote-controlled car with both WiFi and Bluetooth control, plus sensor integration.

## Overview
This project demonstrates wireless control of a hardware vehicle using two communication protocols (WiFi and Bluetooth), along with sensor-based features.

## Tech Stack
- Arduino (C/C++)
- WiFi module
- Bluetooth module
- Sensors

## Files
- `WiFi Car Project Code.*` — WiFi-controlled car implementation
- `WiFi Car Sensor Project Code.*` — sensor-integrated version
- `bluetooth-rc-car-arduino-program.jpg`, `Wifi-controlled-car.jpg` — hardware/circuit images

## How to Run
1. Review the circuit diagrams (included images)
2. Upload the relevant `.ino`/code file to your Arduino board
3. Connect via WiFi or Bluetooth as documented
